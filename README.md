# CarScope — India's Car Discovery Platform

A fully functional replica of CarDekho.com built with vanilla HTML, CSS & JavaScript.

## Files
- `index.html` — Complete website (single file, all code included)
- `carscope-logo.svg` — Full horizontal logo (248×60)
- `carscope-icon.svg` — Square favicon/icon (32×32)

## How to Run
Simply open `index.html` in any modern web browser. No server or dependencies needed.

## Features
- New Cars (60+ models, 6 categories: SUV, Hatchback, Sedan, Luxury, EV, MUV)
- Used Cars marketplace with live filters
- Electric Cars section
- Car Compare tool (side-by-side specs table)
- EMI Calculator
- Car Valuation (3-step sell funnel)
- Car Insurance (compare 7 insurers)
- Car Loan (compare 10 banks)
- Find Dealers (1200+ listings)
- EV Charging Stations map
- Fuel Prices (live city data)
- News & Reviews portal
- Login / Register forms
- Real car images via Wikipedia API
- Real brand logos via Clearbit API

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript (ES6+)
- Wikipedia REST API (car images)
- Clearbit Logo API (brand logos)
- Google Fonts (Plus Jakarta Sans, Bebas Neue)
- No frameworks, no npm, no build step required
